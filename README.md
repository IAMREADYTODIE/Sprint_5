# Проект 5-го спринта. Тестирование Stellar Burgers

## Описание
Автотесты для веб-сервиса Stellar Burgers.
Проверяют регистрацию, авторизацию, переходы в личный кабинет и работу конструктора бургеров.

## Структура проекта
- tests/ - папка с тестами
- locators.py - файл с локаторами элементов
- data.py - тестовые данные (URL, константы)
- helpers.py - вспомогательные функции (генераторы данных)
- conftest.py - фикстуры для запуска браузера

## Установка зависимостей
```bash
pip install -r requirements.txt
```
## Запуск тестов
```bash
pytest
```
